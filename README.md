# check-worker-job-test
Тестовое задание для вакансии .NET-разработчика

Требовалось реализовать WCF-сервис, имеющий 2 метода: один из них принимает чеки и кладёт их в БД MSSQL, второй возвращает указанное количество чеков из БД.
Реализация работы с БД подразумевалась фейковая.
Также требовалось реализовать Windows-службу, следящую за каталогом и реагирующее на появление файла .txt в папке. В файле ожидается json, описывающий чек, и если он валиден - нужно передать его в 1 метод WCF-сервиса.
Модель чека:
•	Guid Id
•	string Number
•	decimal Summ
•	decimal Discount
•	string[] Articles

Логгирование WIN-службы реализовано через log4net, работа с JSON - Newtosoft.JSON, подключение библиотек через NuGet, работа с БД через Dapper.
